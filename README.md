# Factbird Customer Utilities

This repository contains utilities and tools that Factbird offers to its
customers to help with various operational tasks and data management.

## Contents

### Excel Templates

- **`excel/shift-import-sheet.xlsx`** - Template for importing shift data into the Factbird system

## Getting Started

Download the appropriate utility for your needs and follow the instructions
provided with each tool.

For questions or support, please contact Factbird support at [support@factbird.com](support@factbird.com).
